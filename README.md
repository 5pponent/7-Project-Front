# 7팀 웹 프로젝트 [FrontEnd]

## Description

>다른 유저들과 일상을 공유하고 소통이 가능한 SNS 어플리케이션 입니다.

## Functions

- 로그인 / 회원가입 / 회원탈퇴
- 프로필 수정
- 게시글 작성 / 수정 / 삭제
- 댓글 및 대댓글 작성 / 수정 / 삭제
- 유저 팔로우

## Techs

`HTML` `CSS` `JavaScript` `React` `Node.js` `Context API`

## Libraries

* React
* axios
* material-ui


## Develop

`npm install` `npm start`
